## 功能
- 支持自定义协议与应用服务器交互
- 支持HTTP登录验证
- 支持RTMP传输协议
- 支持x264视频编解码
- 支持aac音频编解码
- 支持lua进行会话管理
- 支持视频录制(只录制关键帧,生成mp4)
- 支持WEB页面实时监控(只监控关键帧,生成jpg)


## 性能
* 生产环境支持40个会话(80个推拉流)
